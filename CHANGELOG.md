# Changelog

## v0.1.0 - Initial portable release

### Added

- Docker Compose source-build workflow
- portable `.env`-driven host path configuration
- OMDb provider support
- optional IMDb dataset SQLite fallback
- MakeMKV generic filename detection
- stability wait before processing
- `_Unmatched` safety routing
- JSON sidecar audit files
- Cyberdream web dashboard
- CLI commands: `watch`, `scan`, `once`, `identify`, `imdb-index`
- documentation for install, configuration, workflow, troubleshooting, architecture, and SOP

### Notes

DiscoBall is a post-processing organizer only. It does not rip, decrypt, download, or distribute media.
